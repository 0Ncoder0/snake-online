# 贪吃蛇联机版

+ 安装依赖 `npm install`
+ 编译打包 `npm run build`
+ 启动服务端 `npm run server`
+ 启动客户端 `npm run client`


## 客户端操作

在控制台输入以下代码

+ 加入房间 `client.join("123456")` # "123456" 为双方约定的房间号
+ 开始游戏 `client.start()` # 任意一方输入此代码都会启动游戏
+ 操作方式：使用 WASD 控制蛇身转向
